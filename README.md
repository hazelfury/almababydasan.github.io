# Portfolio

Personal portfolio site for Alma Babydasan, software engineer.

**Live site:** https://hazelfury.github.io/almababydasan.github.io/

## About

This is a single-page portfolio built to showcase full-stack development, machine learning, and ERP systems consulting work while job searching for full-time software engineering roles.

## Sections

- **About** — background summary and key stats
- **Skills** — full-stack development, cloud & DevOps, machine learning, ERP consulting, databases & APIs, data automation
- **Projects** — featured work including the UGenome Training Platform, AppBlaster, and other tooling
- **Tech Stack** — core tools and technologies
- **Education & Certifications** — MS in Computer Science (UT Arlington) and Anthropic certifications
- **Experience** — career timeline from Godrej Infotech through current role
- **Contact** — email, LinkedIn, and a contact form

## Built with

Plain HTML, CSS, and JavaScript. No build step or dependencies — self-contained in a single file with Google Fonts loaded via CDN.

## Running locally

Just open `index.html` in a browser, or serve it with any static file server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
